# Hospital Management System (HMS)

### 🏥 Project Overview
This is a **Full-Stack Web Application** developed to automate hospital workflows. It provides a structured way to manage doctors, patients, and appointment scheduling.

### 🚀 Key Features
* **Admin Dashboard**: For overall management of hospital operations.
* **Doctor Management**: Automated **Doctor ID generation** and onboarding.
* **Appointment System**: Real-time scheduling with **Spring Data JPA** relationships.
* **Digital Receipts**: Generation and printing of payment receipts for patients.

### 🛠 Tech Stack
* **Backend**: Java, Spring Boot, Spring Data JPA, MVC Architecture.
* **Frontend**: Thymeleaf, HTML5, CSS3, JavaScript.
* **Database**: MySQL.
* **Tools Used**: Eclipse IDE, Postman, Git & GitHub.

### 💻 Core Logic Applied
* Used **One-to-Many** mappings between Doctors and Appointments.
* Implemented **RESTful Controllers** for seamless data communication.
* Integrated **Hibernate** for database persistence and CRUD operations.
